# World Happiness Visualization Assignment

This repository contains a data visualization-based analysis of the World Happiness dataset covering the years 2015 to 2019.

## Overview
The objective of this assignment is to analyze global happiness trends and examine the impact of various socio-economic factors such as GDP per capita, social support, health (life expectancy), freedom, generosity, and corruption on overall well-being.

## Scope of Analysis
The notebook includes:
- Trend analysis of average happiness scores over time
- Examination of relationships between key variables using scatter plots and regression
- Regional comparisons using confidence intervals
- Distribution analysis using histograms and density plots
- Multi-plot visualizations for combined insights
- Correlation analysis and clustering of variables
- Regional and temporal heatmap analysis

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Repository Contents
- Jupyter Notebook containing complete analysis (Q1–Q10)
- Dataset files for the years 2015 to 2019

## Conclusion
The analysis demonstrates that happiness is influenced by a combination of economic, social, and health-related factors. While global trends remain relatively stable over time, significant regional disparities persist, highlighting the importance of balanced development across multiple dimensions of well-being.
